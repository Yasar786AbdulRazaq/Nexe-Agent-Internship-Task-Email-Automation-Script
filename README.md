Project: Gmail Automation Script

How to run this project:
1. Clone or download this repository.
2. Install the required Python packages using:
   pip install google-auth google-auth-oauthlib google-api-python-client schedule
3. Download your own 'credentials.json' from Google Cloud Console and place it in the project root folder.
4. Run the script using: python email_automation.py

Files included:
- email_automation.py (Main Python script)
- email_history_log.txt (Sample local logs)
